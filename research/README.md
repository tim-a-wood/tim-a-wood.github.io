# Research

Research artifacts, knowledge library, and the Research Dashboard for the MV / Sprite Workbench project.

## Maintained by
**Research Agent** (`.claude/agents/research.md`) — runs scans, writes reports, maintains this library.

## Contents

| Path | Purpose |
|------|---------|
| `dashboard.md` | Agent-readable current status: open issues, opportunities, recent reports |
| `library/INDEX.md` | Master catalog of all library documents — **search here first** |
| `library/findings/` | Raw codebase scan findings |
| `library/reports/` | Synthesized research reports |
| `library/technical/` | Technical assessments and capability research |
| `library/competitive/` | Competitive analysis and market intelligence |

## Dashboard

The Research view is integrated into the Agent OS dashboard (`os-dashboard.html` → Research in the left nav).

## Filing Convention

All library documents use the naming pattern: `[topic]-[YYYY-MM-DD].md`

Example: `codebase-scan-2026-04-01.md`, `sprite-editor-competitive-2026-04-01.md`
