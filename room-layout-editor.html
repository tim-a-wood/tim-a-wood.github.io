Phase 1 · Layout

Layout

Name the room, set its id, define the footprint polygon, then add walkable platforms (axis-aligned rects for Phaser Arcade) with the canvas tools or presets below.

Identity

Display name and canonical id are used in the editor, map graph, and exports.

Footprint

Preset sizes match common viewports; custom uses your width and height in pixels.

Pick a preset or custom dimensions. Changing footprint rescales room-local coordinates — review doors and platforms after large changes.

Neighbors & alignment

Connect this room to another, align edges, optionally match wall length (rectangle rooms), then match opening along the wall.

Pick a neighboring room and which edges meet. Align snaps the shared edges flush in world space (perpendicular only, no doors required). Match opening height slides along that wall (doors or edge midpoints). Match wall length resizes this room’s rectangle footprint so the selected edge length (local px) matches the neighbor’s edge — use for axis-aligned rectangle rooms only; recheck doors/platforms after. With only one room in the project, add another room first.

Preset library

Appends platforms inside the room footprint (unlocks when name, id, and footprint are valid). Adjust or delete on the canvas after placing.

Platform selection

Duplicate uses editor snap step for horizontal offset. Mirror clones across room horizontal center.

Door / platform checks

Warns when a door anchor sits inside a platform’s walkable band.

Complete layout (name, id, footprint) to refresh checks.
