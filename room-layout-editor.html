Room Layout Editor

Edit room floorplans, ledges, doors, and global placement. During development, use git as the shared cross-device sync mechanism.
Development Workflow
  1. Choose a room, edit the layout, then tap Save Device. This saves the current layout in this browser on this device.
  2. Tap Export Layout from Advanced when you want to move the current layout to another device or commit it to git.
  3. On your MacBook, replace [room-layout-data.json](/Users/timwood/Desktop/projects/PWA/MV/room-layout-data.json) with the exported file, then `git add`, `git commit`, and `git push`.
  4. On another device, pull/reload the latest repo version, or use Import Layout from Advanced to paste the JSON directly.
  5. Tap Open Game to test the exact current editor state immediately in the same tab.
  6. Use Reload From Disk only when you explicitly want to discard the current device-saved layout and reload [room-layout-data.json](/Users/timwood/Desktop/projects/PWA/MV/room-layout-data.json).
